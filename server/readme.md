## Dependencias 

1º - npm i typescript -D
2º - npx tsc --init
3º - npm i tsx -D
4º - npm i -d prisma
5º - npm i @prisma/client
6º - npm i @fastify/cors

## Database 
1º - npx prisma init --datasource-provider SQLite
2º - npx prisma migrate dev

## RUN
1º - npx tsx src/server.ts
2º - npx prisma studio
